Projekt 1 des CAS FEE.

## Aufsetzen

```bash
# Repo klonen
git clone https://github.com/TheMen4ce/cas-fee_01.git

# In Verzeichnis wechseln
cd cas-fee_01

# NPM-Dependencies installieren (in package.json definiert)
npm i
```

## Webserver starten

```bash
npm start
```

Im Browser unter http://localhost:3000 erreichbar.
